# DigiSpace Dashboard UI

A React based dashboard created from the provided design.

Resolution Used
1440 × 900

Tech Stack
ReactJS
Redux Toolkit
Apache ECharts
CSS Grid

Features
• Lights energy gauge
• Water consumption card
• Carbon intensity donut chart
• Energy consumption bar chart
• Footfall line chart

Architecture
Each card is implemented as an independent reusable component.

Extra Credit Implemented
✔ Adaptive Layout using CSS Grid
✔ Redux State Management
✔ Hosted on Vercel

Live Application on vercel
https://digispace-dashboard.vercel.app

Installation

Clone the repository:

git clone https://github.com/poojathakral97/dashboard-ui.git

Navigate into the project:

cd dashboard-ui

Install dependencies:

npm install

Run the project:

npm start
